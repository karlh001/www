Overview


A complete and recoverable back-up plan is essential to protecting your digital assets. All data storage devices will eventually die, so it's important to have multiple copies of important data. I follow the 3-2-1 rule for all my data. Here's how I do it:

  1. Primary data (laptop/phone/desktop)
  2. Windows File History to external hard drive / Macrium (full disk image and documents) back-up to external hard drive
  3. Cloud back-up to Mega and AWS S3
  4. Back-up to network attached storage, running ZFS to maintain data integrity
  5. Network attached storage server back-up to external hard drive
  6. DVD archival for long-term back-up / storage of important data

So once a file has been through the back-up process, it will be copied 6 times!

A back up is only good if you can restore it. This is an often forgotten task. You need to practice restoring files and see if they are readable.
